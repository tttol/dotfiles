# Markdown to HTML Converter
This skill converts Markdown files into HTML. Markdown generated by AI tools (such as OpenAI Codex or Claude Code) is often too massive to read comfortably, even if the content itself is flawless. Realistically, humans don't read huge files line by line—we prefer scannable content that leverages visuals like graphs, tables, and diagrams.

## Use Case
- Processing oversized Markdown files generated by AI coding assistants.

## Key Features
- **Consistency:** Converts text flawlessly without altering the original content.
- **Human-Readable:** Transforms dense walls of text into a clean, human-friendly layout.
- **Visual-First:** Emphasizes rich visual elements, including graphs, tables, and diagrams.
